# Docs

当前文档只保留正在作为设计、开发、验证依据的材料。

## 当前入口

- `stability-mechanisms.md`：项目长期稳定性机制说明。
- `测试问题记录清单.md`：近期用户视角压测问题记录。
- `superpowers/`：当前有效的设计文档与实现计划。
- `archive/`：历史需求、阶段稿、评审记录、截图和过期计划。

## 目录规则

- 顶层不新增临时目录。
- 早期需求、一次性记录、阶段性设计、旧截图统一进 `archive/`。
- 新功能设计放 `superpowers/specs/`。
- 已确认的执行计划放 `superpowers/plans/`。
- 被新版覆盖或已失效的 `superpowers` 文档移到 `archive/superpowers/`。
