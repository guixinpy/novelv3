## 三、Hermes 写作对话（`/projects/:id/hermes`）

- [x] **3.1 创作阶段** — SubNav 显示"创作阶段"区域，包含四步进度条：设定 → 故事线 → 大纲 → 正文，当前阶段高亮
- [x] **3.2 章节列表** — SubNav 显示"章节"区域，列出已有章节（如有），点击章节切换
- [ ] **3.3 导出按钮** — SubNav 底部有"导出"按钮
- [ ] **3.4 版本历史按钮** — SubNav 底部有"版本历史"按钮

### 3B. 聊天核心功能

- [ ] **3.5 消息历史** — 聊天区域加载并显示历史消息，助手消息左侧有品牌色边框，用户消息右对齐
- [ ] **3.6 发送普通消息** — 在输入框输入"你好"，按 Enter 发送，显示用户消息气泡，等待助手回复
- [ ] **3.7 加载状态** — 发送消息后，输入框禁用，显示加载动画（三个跳动的点）
- [ ] **3.8 自动滚动** — 新消息出现时，聊天区域自动滚动到底部

### 3C. 斜杠命令

- [ ] **3.9 命令菜单触发** — 输入 `/`，弹出命令菜单，显示所有可用命令（clear、compact、setup、storyline、outline）
- [ ] **3.10 命令过滤** — 输入 `/s`，菜单只显示 setup 和 storyline
- [ ] **3.11 /setup 命令** — 输入 `/setup 主角是一个剑客`，发送后 AI 返回世界设定预览，出现确认/取消/修改按钮
- [ ] **3.12 确认动作** — 点击"确认"按钮，设定被保存，创作阶段进度更新
- [ ] **3.13 取消动作** — 再次执行 `/setup`，这次点击"取消"，动作被取消，无副作用
- [ ] **3.14 修改动作** — 再次执行 `/setup`，点击"修改"，输入修改意见，AI 重新生成
- [ ] **3.15 /storyline 命令** — 输入 `/storyline`，AI 生成故事线预览，确认后保存
- [ ] **3.16 /outline 命令** — 输入 `/outline`，AI 生成章节大纲预览，确认后保存
- [ ] **3.17 /clear 命令** — 输入 `/clear`，聊天历史被清空
- [ ] **3.18 /compact 命令** — 先发几条消息，然后输入 `/compact`，历史被压缩为摘要

### 3D. 自然语言意图

- [ ] **3.19 自然语言生成设定** — 输入"帮我生成世界设定"，AI 识别意图并返回设定预览（与 /setup 效果相同）
- [ ] **3.20 查询诊断** — 输入"接下来做什么"，AI 返回当前项目状态诊断和建议

### 3E. 导出与版本

- [ ] **3.21 导出模态框** — 点击 SubNav "导出"按钮，弹出模态框，显示 Markdown 和 TXT 两个选项
- [ ] **3.22 执行导出** — 选择 Markdown 格式，点击导出，文件下载成功
- [ ] **3.23 版本历史模态框** — 点击"版本历史"按钮，弹出模态框，显示版本记录表格
- [ ] **3.24 版本筛选** — 在版本模态框中按类型筛选（设定/故事线/大纲等）
- [ ] **3.25 版本回滚** — 选择一个历史版本，点击回滚，确认后数据恢复到该版本

## 七、完整创作流程（端到端）

> 使用新建项目，从零走完整个创作流程

- [ ] **7.1 新建项目** — 在首页创建项目"端到端测试"，类型"奇幻"
- [ ] **7.2 进入 Hermes** — 点击进入项目，确认处于"设定"阶段
- [ ] **7.3 生成设定** — 输入 `/setup 一个关于龙与魔法的奇幻世界`，确认生成的设定
- [ ] **7.4 查看 Athena 设定** — 切换到 Athena，确认角色/地点/规则等本体数据已填充
- [ ] **7.5 生成故事线** — 切回 Hermes，输入 `/storyline`，确认生成的故事线
- [ ] **7.6 阶段进度更新** — 确认创作阶段进度条更新到"故事线"
- [ ] **7.7 生成大纲** — 输入 `/outline`，确认生成的章节大纲
- [ ] **7.8 章节列表更新** — 确认 SubNav 章节列表出现章节条目
- [ ] **7.9 查看 Athena 时间线** — 切换到 Athena → 时间线，确认有事件数据
- [ ] **7.10 开始写作** — 切回 Hermes，输入"开始写第1章"或通过 UI 触发写作
- [ ] **7.11 正文生成** — 等待 AI 生成第1章正文，确认章节内容出现
- [ ] **7.12 查看 Manuscript** — 切换到 Manuscript，确认章节列表有第1章
