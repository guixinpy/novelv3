## 1. 概述

一致性保障采用**渐进式实施策略**：

| 阶段 | 功能 | 说明 |
|------|------|------|
| **MVP** | L1 规则提取 + 基础冲突检测 | 满足 80% 的一致性需求 |
| **成长** | 引入 L2 LLM 提取 + 交叉验证 | 提升复杂场景检测能力 |
| **成熟** | 完整 7 类检查器 + 时间线管理 | 达到生产级一致性保障 |

## 3. 双层事实提取系统

### 3.1 设计思想

| 层级 | 方法 | 优点 | 缺点 | 使用场景 |
|------|------|------|------|----------|
| **L1** | 规则/正则 | 速度快、成本低、确定性高 | 只能提取结构化事实 | 实时检查、大批量处理 |
| **L2** | LLM 语义 | 能提取复杂隐含事实 | 速度慢、成本高 | 关键章节、复杂场景 |
| **交叉验证** | L1+L2 | 高置信度、互补优势 | 成本最高 | 重要一致性检查 |

### 3.2 L1: 规则提取器

```typescript
class L1RuleExtractor {
  private patterns: ExtractPattern[] = [
    // 角色死亡检测
    {
      type: 'character_state_change',
      regex: /(\S+)(?:已经|早已)?死[了亡]/g,
      extractor: (match) => ({
        subject: match[1],
        attribute: 'state',
        newValue: 'dead',
      }),
    },
    // 角色突破检测
    {
      type: 'character_ability_change',
      regex: /(\S+)(?:突破|晋升|达到)到?\s*(炼气|筑基|金丹|元婴|化神|渡劫|大乘)/g,
      extractor: (match) => ({
        subject: match[1],
        attribute: 'realm',
        newValue: match[2],
      }),
    },
    // 物品转移检测
    {
      type: 'item_ownership_change',
      regex: /(\S+)(?:交给|送给|赠给)了?(\S+)/g,
      extractor: (match) => ({
        subject: match[2],
        attribute: 'owner',
        newValue: match[1],
      }),
    },
  ];

  extract(chapterContent: string, chapterIndex: number): ExtractedFact[] {
    const facts: ExtractedFact[] = [];

    for (const pattern of this.patterns) {
      let match;
      while ((match = pattern.regex.exec(chapterContent)) !== null) {
        const data = pattern.extractor(match);
        facts.push({
          id: generateUUID(),
          type: pattern.type as FactType,
          source: 'l1_rule',
          confidence: 0.85,
          extractedAt: Date.now(),
          chapterIndex,
          data,
          evidence: {
            text: match[0],
            position: this.estimatePosition(chapterContent, match.index),
          },
          validation: {
            l1Extracted: true,
            crossConfidence: 0.85,
          },
        });
      }
    }

    return facts;
  }

  private estimatePosition(content: string, index: number) {
    const before = content.slice(0, index);
    const paragraphs = before.split('\n').length;
    return { paragraph: paragraphs, sentence: 0 };
  }
}
```

```typescript
class L2LLMExtractor {
  constructor(private aiService: AIService) {}

  async extract(chapterContent: string, chapterIndex: number): Promise<ExtractedFact[]> {

请从以下小说章节中提取关键事实，以 JSON 数组格式返回。

需要提取的事实类型：
- character_state_change: 角色状态变化（死亡、受伤、昏迷等）
- character_ability_change: 角色能力变化（突破境界、学会技能等）
- character_emotion_change: 角色情绪变化
- item_ownership_change: 物品所有权变化
- location_change: 地点变化
- relationship_change: 关系变化
- foreshadow_planted: 埋下伏笔
- foreshadow_resolved: 回收伏笔

章节内容：
${chapterContent.slice(0, 4000)}

返回格式：

  {
    "type": "fact_type",
    "subject": "主体名称",
    "attribute": "属性",
    "oldValue": "原值（可选）",

    "evidence": "原文证据",
    "confidence": 0.9
  }
]
`;

    try {
      const response = await this.aiService.complete(prompt, {
        temperature: 0.3,
        maxTokens: 2000,

      const rawFacts = JSON.parse(response);
      return rawFacts.map((fact: any) => ({
        id: generateUUID(),
        type: fact.type,

        },

      console.error('L2 extraction failed:', error);

    }
  }
}
```

### 3.4 交叉验证机制

| 情况 | 处理方式 | 最终置信度 |
|------|----------|------------|
| L1 和 L2 都提取到 | 合并，增强置信度 | `min(0.95, max(L1, L2) + 0.1)` |
| 只有 L1 提取到 | 标记为待确认 | `L1 置信度 × 0.6` |
| 只有 L2 提取到 | 标记为待确认 | `L2 置信度 × 0.7` |
| L1 和 L2 冲突 | 生成冲突告警 | 需人工审核 |

```typescript
class CrossValidator {
  validate(l1Facts: ExtractedFact[], l2Facts: ExtractedFact[]): CrossValidationResult {
    const confirmed: ExtractedFact[] = [];
    const conflicts: FactConflict[] = [];
    const pending: ExtractedFact[] = [];

    const l1Map = new Map(l1Facts.map((f) => [this.factKey(f), f]));
    const l2Map = new Map(l2Facts.map((f) => [this.factKey(f), f]));

    // 处理 L1 和 L2 都提取到的事实
    for (const [key, l1Fact] of l1Map) {
      const l2Fact = l2Map.get(key);
      if (l2Fact) {
        const merged = this.mergeFacts(l1Fact, l2Fact);
        confirmed.push(merged);
        l2Map.delete(key);
      } else {
        pending.push({
          ...l1Fact,
          validation: {
            l1Extracted: true,
            crossConfidence: l1Fact.confidence * 0.6,
          },
        });
      }
    }

    // 处理仅 L2 提取到的事实
    for (const l2Fact of l2Map.values()) {
      pending.push({
        ...l2Fact,
        validation: {
          l2Extracted: true,
          crossConfidence: l2Fact.confidence * 0.7,
        },
      });
    }

    return {
      confirmed,
      conflicts,
      pending,
      stats: {
        l1Count: l1Facts.length,
        l2Count: l2Facts.length,
        confirmedCount: confirmed.length,
        conflictCount: conflicts.length,
        pendingCount: pending.length,
      },
    };
  }

  private factKey(fact: ExtractedFact): string {
    return `${fact.type}:${fact.data.subject}:${fact.data.attribute}:${fact.data.newValue}`;
  }

  private mergeFacts(l1: ExtractedFact, l2: ExtractedFact): ExtractedFact {
    return {
      ...l1,
      confidence: Math.min(0.95, Math.max(l1.confidence, l2.confidence) + 0.1),
      validation: {
        l1Extracted: true,
        l2Extracted: true,
        crossConfidence: Math.min(0.95, Math.max(l1.confidence, l2.confidence) + 0.1),
      },
    };
  }
}
```

## 4. 冲突检测器

### 4.1 7 类专门检查器

```typescript
interface ConsistencyChecker {
  check(facts: ExtractedFact[], topology: Topology): ConsistencyIssue[];
}

// 1. 角色状态检查器
class CharacterStateChecker implements ConsistencyChecker {
  check(facts: ExtractedFact[]): ConsistencyIssue[] {
    const issues: ConsistencyIssue[] = [];
    const characterStates = new Map<string, { state: string; chapter: number }>();

    for (const fact of facts.filter((f) => f.type === 'character_state_change')) {
      const current = characterStates.get(fact.data.subject);

      if (current) {
        // 死亡后复活检测
        if (current.state === 'dead' && fact.data.newValue !== 'dead') {
          issues.push({
            id: generateUUID(),
            type: 'CHARACTER_RESURRECTION',
            severity: 'fatal',
            subject: fact.data.subject,
            description: `角色 ${fact.data.subject} 在第 ${current.chapter} 章已死亡，但在第 ${fact.chapterIndex} 章状态变为 ${fact.data.newValue}`,
            detectedAt: Date.now(),
            checkerName: 'CharacterStateChecker',
          });
        }
      }

      characterStates.set(fact.data.subject, {
        state: fact.data.newValue,
        chapter: fact.chapterIndex,
      });
    }

    return issues;
  }
}

// 2. 角色能力检查器
class CharacterAbilityChecker implements ConsistencyChecker {
  private realmOrder = ['炼气', '筑基', '金丹', '元婴', '化神', '渡劫', '大乘'];

  check(facts: ExtractedFact[]): ConsistencyIssue[] {
    const issues: ConsistencyIssue[] = [];
    const characterRealms = new Map<string, string>();

    for (const fact of facts.filter((f) => f.type === 'character_ability_change')) {
      if (fact.data.attribute === 'realm') {
        const currentRealm = characterRealms.get(fact.data.subject);
        if (currentRealm) {
          const currentIndex = this.realmOrder.indexOf(currentRealm);
          const newIndex = this.realmOrder.indexOf(fact.data.newValue);
          if (newIndex < currentIndex) {
            issues.push({
              id: generateUUID(),
              type: 'REALM_REGRESSION',
              severity: 'warn',
              subject: fact.data.subject,
              description: `角色 ${fact.data.subject} 境界从 ${currentRealm} 倒退到 ${fact.data.newValue}`,
              detectedAt: Date.now(),
              checkerName: 'CharacterAbilityChecker',
            });
          }
        }
        characterRealms.set(fact.data.subject, fact.data.newValue);
      }
    }

    return issues;
  }
}

// 3. 物品所有权检查器
class ItemOwnershipChecker implements ConsistencyChecker {
  check(facts: ExtractedFact[]): ConsistencyIssue[] {
    const issues: ConsistencyIssue[] = [];
    const itemOwners = new Map<string, { owner: string; chapter: number }>();

    for (const fact of facts.filter((f) => f.type === 'item_ownership_change')) {
      const current = itemOwners.get(fact.data.subject);
      if (current && current.owner !== fact.data.newValue) {
        issues.push({
          id: generateUUID(),
          type: 'OWNERSHIP_CONFLICT',
          severity: 'warn',
          subject: fact.data.subject,
          description: `物品 ${fact.data.subject} 所有权从 ${current.owner} 变为 ${fact.data.newValue}，但未检测到转移过程`,
          detectedAt: Date.now(),
          checkerName: 'ItemOwnershipChecker',
        });
      }
      itemOwners.set(fact.data.subject, { owner: fact.data.newValue, chapter: fact.chapterIndex });
    }

    return issues;
  }
}

// 4. 时间线检查器
class TimelineChecker implements ConsistencyChecker {
  check(facts: ExtractedFact[]): ConsistencyIssue[] {
    const issues: ConsistencyIssue[] = [];
    let currentTime = 0;

    for (const fact of facts
      .filter((f) => f.type === 'time_explicit' || f.type === 'time_relative')
      .sort((a, b) => a.chapterIndex - b.chapterIndex)) {
      const timeValue = parseInt(fact.data.newValue, 10);
      if (!isNaN(timeValue) && timeValue < currentTime) {
        issues.push({
          id: generateUUID(),
          type: 'NEGATIVE_TIME',
          severity: 'warn',
          subject: 'timeline',
          description: `第 ${fact.chapterIndex} 章出现时间倒退：${timeValue} < ${currentTime}`,
          detectedAt: Date.now(),
          checkerName: 'TimelineChecker',
        });
      }
      if (!isNaN(timeValue)) {
        currentTime = timeValue;
      }
    }

    return issues;
  }
}

// 5. 地点位置检查器
class LocationChecker implements ConsistencyChecker {
  check(facts: ExtractedFact[], topology: Topology): ConsistencyIssue[] {
    const issues: ConsistencyIssue[] = [];
    const characterLocations = new Map<string, { location: string; chapter: number }>();

    for (const fact of facts.filter((f) => f.type === 'location_change')) {
      const current = characterLocations.get(fact.data.subject);
      if (current && current.location !== fact.data.newValue) {
        issues.push({
          id: generateUUID(),
          type: 'LOCATION_JUMP',
          severity: 'info',
          subject: fact.data.subject,
          description: `角色 ${fact.data.subject} 从 ${current.location} 移动到 ${fact.data.newValue}`,
          detectedAt: Date.now(),
          checkerName: 'LocationChecker',
        });
      }
      characterLocations.set(fact.data.subject, {
        location: fact.data.newValue,
        chapter: fact.chapterIndex,
      });
    }

    return issues;
  }
}

// 6. 关系检查器
class RelationshipChecker implements ConsistencyChecker {
  check(facts: ExtractedFact[]): ConsistencyIssue[] {
    const issues: ConsistencyIssue[] = [];
    const relationships = new Map<string, string>();

    for (const fact of facts.filter((f) => f.type === 'relationship_change')) {
      const key = [fact.data.subject, fact.data.attribute].sort().join('-');
      const current = relationships.get(key);
      if (current && current !== fact.data.newValue) {
        issues.push({
          id: generateUUID(),
          type: 'RELATIONSHIP_MUTATION',
          severity: 'info',
          subject: fact.data.subject,
          description: `关系突变：${fact.data.subject} 与 ${fact.data.attribute} 从 ${current} 变为 ${fact.data.newValue}`,
          detectedAt: Date.now(),
          checkerName: 'RelationshipChecker',
        });
      }
      relationships.set(key, fact.data.newValue);
    }

    return issues;
  }
}

// 7. 伏笔检查器
class ForeshadowChecker implements ConsistencyChecker {
  check(facts: ExtractedFact[], topology: Topology): ConsistencyIssue[] {
    const issues: ConsistencyIssue[] = [];
    const fsNodes = topology.nodes.filter((n) => n.type === 'FORESHADOWING');

    for (const node of fsNodes) {
      const setupChapter = node.data.foreshadowingId
        ? parseInt(node.metadata.firstAppearance as any, 10)
        : 0;
      const payoffChapter = node.data.foreshadowingId
        ? parseInt(node.metadata.lastAppearance as any, 10)
        : 0;

      if (node.data.foreshadowingStatus !== 'PAYOFF' && payoffChapter > 0) {
        issues.push({
          id: generateUUID(),
          type: 'FORESHADOW_UNRESOLVED',
          severity: 'warn',
          subject: node.label,
          description: `伏笔 "${node.label}" 应在第 ${payoffChapter} 章回收，但尚未回收`,
          detectedAt: Date.now(),
          checkerName: 'ForeshadowChecker',
        });
      }
    }

    return issues;
  }
}
```

### 4.2 检查器执行流程

```typescript
class EnhancedConflictDetector {
  private checkers: ConsistencyChecker[] = [
    new CharacterStateChecker(),
    new CharacterAbilityChecker(),
    new ItemOwnershipChecker(),
    new TimelineChecker(),
    new LocationChecker(),
    new RelationshipChecker(),
    new ForeshadowChecker(),
  ];

  async detect(
    newFacts: ExtractedFact[],
    existingFacts: ExtractedFact[],
    topology: Topology
  ): Promise<ConsistencyIssue[]> {
    const allFacts = [...existingFacts, ...newFacts];

    // 并行执行所有检查器
    const results = await Promise.all(
      this.checkers.map((checker) => checker.check(allFacts, topology))
    );

    // 合并并去重
    const allIssues = results.flat();
    const deduped = this.deduplicateIssues(allIssues);

    // 按严重程度排序
    const severityOrder = { fatal: 0, warn: 1, info: 2 };
    return deduped.sort((a, b) => severityOrder[a.severity] - severityOrder[b.severity]);
  }

  private deduplicateIssues(issues: ConsistencyIssue[]): ConsistencyIssue[] {
    const seen = new Set<string>();
    return issues.filter((issue) => {
      const key = `${issue.type}:${issue.subject}:${issue.description}`;
      if (seen.has(key)) return false;
      seen.add(key);
      return true;
    });
  }
}
```

## 5. 一致性引擎

```typescript
class ConsistencyEngine {
  constructor(
    private storage: Storage,
    private aiService: AIService,
    private topologyManager: TopologyManager,
    private l1Extractor: L1RuleExtractor,
    private l2Extractor: L2LLMExtractor,
    private crossValidator: CrossValidator,
    private conflictDetector: EnhancedConflictDetector
  ) {}

  async preCheck(projectId: string, chapterIndex: number): Promise<ConsistencyIssue[]> {
    const outline = await this.storage.getOutline(projectId);
    const topology = await this.storage.getTopology(projectId);
    const issues: ConsistencyIssue[] = [];

    const chapterOutline = outline.chapters[chapterIndex - 1];

    // MVP: 基础角色状态检查
    for (const characterId of chapterOutline.characters) {
      const charNode = topology.nodes.find(
        (n) => n.type === 'CHARACTER' && n.data.characterId === characterId
      );
      if (charNode) {
        const deathEvent = topology.nodes.find(
          (n) =>
            n.type === 'EVENT' &&
            n.data.eventDescription?.includes('死亡') &&
            n.data.eventChapter < chapterIndex
        );
        if (deathEvent) {
          issues.push({
            id: generateUUID(),
            type: 'CHARACTER_STATE',
            severity: 'fatal',
            subject: charNode.label,
            description: `角色 ${charNode.label} 在第 ${deathEvent.data.eventChapter} 章已死亡，不能出现在本章`,
            detectedAt: Date.now(),
            checkerName: 'ConsistencyEngine',
          });
        }
      }
    }

    return issues;
  }

  async postCheck(
    projectId: string,
    chapterIndex: number,
    content: ChapterContent
  ): Promise<ConsistencyIssue[]> {
    const allFacts = await this.storage.getExtractedFacts(projectId);

    // MVP 阶段可只使用 L1；成长/成熟阶段启用 L2 + 交叉验证
    const l1Facts = this.l1Extractor.extract(content.content, chapterIndex);
    const l2Facts = await this.l2Extractor.extract(content.content, chapterIndex);
    const validated = this.crossValidator.validate(l1Facts, l2Facts);

    const topology = await this.storage.getTopology(projectId);
    const issues = await this.conflictDetector.detect(
      validated.confirmed,
      allFacts,
      topology
    );

    // 保存新提取的事实
    for (const fact of validated.confirmed) {
      await this.storage.saveExtractedFact(projectId, fact);
    }

    return issues;
  }

  async buildTopology(projectId: string, outline: Outline): Promise<Topology> {
    const nodes: TopologyNode[] = [];
    const edges: TopologyEdge[] = [];
    const setup = await this.storage.getSetup(projectId);

    for (const character of setup.characters) {
      nodes.push({
        id: `char_${character.id}`,
        type: 'CHARACTER',
        label: character.name,
        data: { characterId: character.id, characterData: character },
        position: { x: 0, y: 0 },
        metadata: { firstAppearance: 0, lastAppearance: 0, appearanceCount: 0 },
      });
    }

    for (const chapter of outline.chapters) {
      for (const location of chapter.locations) {
        const existing = nodes.find((n) => n.type === 'LOCATION' && n.data.locationName === location);
        if (!existing) {
          nodes.push({
            id: `loc_${generateUUID()}`,
            type: 'LOCATION',
            label: location,
            data: { locationName: location },
            position: { x: 0, y: 0 },
            metadata: { firstAppearance: chapter.index, lastAppearance: chapter.index, appearanceCount: 1 },
          });
        } else {
          existing.metadata.lastAppearance = chapter.index;
          existing.metadata.appearanceCount++;
        }
      }

      for (const fsId of chapter.foreshadowingSetup) {
        const fs = outline.foreshadowing.find((f) => f.id === fsId);
        if (fs) {
          nodes.push({
            id: `fs_${fs.id}`,
            type: 'FORESHADOWING',
            label: `伏笔: ${fs.description.slice(0, 20)}...`,
            data: { foreshadowingId: fs.id, foreshadowingStatus: 'SETUP' },
            position: { x: 0, y: 0 },
            metadata: { firstAppearance: fs.setupChapter, lastAppearance: fs.payoffChapter, appearanceCount: 2 },
          });
        }
      }
    }

    for (const character of setup.characters) {
      for (const rel of character.relationships) {
        edges.push({
          id: `edge_${generateUUID()}`,
          source: `char_${character.id}`,
          target: `char_${rel.characterId}`,
          type: 'RELATIONSHIP',
          label: rel.type,
          weight: 1,
          chapters: [],
        });
      }
    }

    const topology: Topology = {
      id: generateUUID(),
      projectId,
      version: 1,
      nodes,
      edges,
      indexes: this.buildIndexes(nodes),
      updatedAt: Date.now(),
    };

    await this.storage.saveTopology(topology);
    return topology;
  }

  async updateTopology(
    projectId: string,
    chapterIndex: number,
    content: ChapterContent
  ): Promise<void> {
    const topology = await this.storage.getTopology(projectId);

    for (const characterId of content.metadata.characters) {
      const charNode = topology.nodes.find(
        (n) => n.type === 'CHARACTER' && n.data.characterId === characterId
      );
      if (charNode) {
        charNode.metadata.lastAppearance = chapterIndex;
        charNode.metadata.appearanceCount++;
      }
    }

    for (const fsId of content.metadata.foreshadowingHandled) {
      const fsNode = topology.nodes.find(
        (n) => n.type === 'FORESHADOWING' && n.data.foreshadowingId === fsId
      );
      if (fsNode) {
        fsNode.data.foreshadowingStatus = 'PAYOFF';
      }
    }

    topology.version++;
    topology.updatedAt = Date.now();
    await this.storage.saveTopology(topology);
  }

  private buildIndexes(nodes: TopologyNode[]): TopologyIndexes {
    const byType: Record<TopologyNodeType, string[]> = {
      CHARACTER: [],
      LOCATION: [],
      EVENT: [],
      ITEM: [],
      PLOTPOINT: [],
      FORESHADOWING: [],
      THEME: [],
    };

    const byChapter: Record<number, string[]> = {};
    const byCharacter: Record<string, string[]> = {};

    for (const node of nodes) {
      byType[node.type].push(node.id);
      if (node.metadata.firstAppearance) {
        for (let i = node.metadata.firstAppearance; i <= node.metadata.lastAppearance; i++) {
          if (!byChapter[i]) byChapter[i] = [];
          byChapter[i].push(node.id);
        }
      }
      if (node.type === 'CHARACTER' && node.data.characterId) {
        byCharacter[node.data.characterId] = [node.id];
      }
    }

    return { byType, byChapter, byCharacter };
  }
}
```
