## 核心理念

墨舟 AI Writer 是一款 **AI 全自动长篇写作系统**，通过单一对话入口，让 AI 自主完成从创意到成品的完整写作流程。

### 关键原则

1. **单一入口** - 对话即一切
2. **透明可控** - 进度可视、审批保留
3. **渐进增强** - MVP 先跑通核心流程，再逐步引入高级功能
4. **长篇保障** - 拓扑图 + 一致性引擎确保长文本质量
