# Longform Scale Phase 46: Athena 正文进度块有限查询

## 目标

Athena 对话会注入正文进度块。当前显示层已经折叠长篇章节清单，但构建阶段仍会加载全部章节摘要行。千章项目中，这会让每次 Athena 对话都承担不必要的数据库和内存成本。本阶段改为聚合统计、开篇有限查询、最新有限查询。

## 验收标准

1. 正文进度仍显示已生成章节数、目标章节数、总字数和章节范围。
2. 40 章以内仍可显示完整章节清单。
3. 长篇项目只查询开篇和最新章节摘要，不做全量章节摘要 `.all()`。
4. 最近章节正文摘录仍只读取最近少量章节。
5. 相关测试、后端测试、前端测试和构建通过。
